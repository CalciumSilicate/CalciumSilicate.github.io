Archlinux下安装Clash

2023/4/5

Clash 是由Go语言实现的跨平台代理工具

官方: https://github.com/Dreamacro/clash/

安装 Clash

Arch Linux下载linux-amd64即可

wget https://github.com/Dreamacro/clash/releases/download/v1.14.0/clash-linux-amd64-v1.14.0.gz
gzip -d clash-linux-amd64-v1.14.0.gz

移动到 /usr/local/bin 目录并给予可执行权限

sudo mv clash-linux-amd64-v1.14.0 /usr/local/bin/clash
sudo chmod +x /usr/local/bin/clash

启动 Clash

clash 启动后会在 ~/.config/clash 目录生成配置文件

clash

远程管理

vim ~/.config/clash/config.yaml

external-controller: '127.0.0.1:9090'

管理：http://clash.razord.top/#/proxies

添加节点

cd ~/.config/clash

# 备份
mv config.yaml config.yaml.backup

wget [订阅地址] -O config.yaml

命令行代理

安装 proxychains

yay -S proxychains

修改配置文件

sudo vim /etc/proxychains.conf

socks5  127.0.0.1 7891

使用

proxychains4 curl www.google.com
